    ```typescript
    export interface ClusterResult {
      clusters: string[][];  // Each cluster is array of node IDs
      noise: string[];       // Node IDs that are outliers (D-07)
    }

    export interface ClusterInfo {
      id: number;               // cluster index
      nodeIds: string[];        // node IDs in this cluster
      centroid: Point;          // geometric center of the cluster
      nodes: CanvasNodeInfo[];  // full node data for narrative building
    }
    ```

    **Functions to implement:**

    1. `computeAdaptiveEpsilon(nodes: CanvasNodeInfo[], multiplier?: number): number`
       - Per D-01 and RESEARCH Pattern 2
       - If nodes.length < 2, return Infinity
       - For each node, find its nearest neighbor distance (using computeCenter + euclideanDistance)
       - Sort all nearest-neighbor distances, take the median
       - Return median * multiplier (default DEFAULT_SPATIAL_CONFIG.adaptiveMultiplier = 2.0)
       - The multiplier of 2.0 ensures conservative clustering per D-06 -- "fewer, larger clusters"

    2. `dbscan(nodes: CanvasNodeInfo[], epsilon: number, minPoints: number): ClusterResult`
       - Standard DBSCAN algorithm from RESEARCH.md Code Example 2
       - Uses computeCenter + euclideanDistance for regionQuery
       - Labels: -1 = unvisited, -2 = NOISE, >= 0 = cluster ID
       - For each unvisited node: find neighbors within epsilon. If neighbors.length < minPoints, mark as NOISE. Otherwise start new cluster and expand.
       - Per D-07: NOISE nodes are outliers, NOT clusters of one
       - Return: `{ clusters: string[][], noise: string[] }`

    3. `findFocusCluster(clusters: ClusterInfo[], triggerNodeId: string, nodes: CanvasNodeInfo[]): ClusterInfo | null`
       - Per D-08: the focus cluster is the one nearest to the most recently edited node
       - Find the trigger node from nodes array by ID
       - If trigger node not found or no clusters, return null
       - Compute distance from trigger node center to each cluster centroid
       - Return the cluster with the shortest distance

    **Helper (not exported):**
    - `computeClusterCentroid(nodeIds: string[], nodes: CanvasNodeInfo[]): Point` -- average of center coordinates of all nodes in the cluster

    **Functions to implement:**

    1. `checkCollision(candidate: BoundingBox, existing: BoundingBox[], gap: number = 40): boolean`
       - Per D-11: expand each existing node's bounding box by `gap` on all sides before checking
       - AABB overlap: `candidate.x < expanded.x + expanded.width && candidate.x + candidate.width > expanded.x && candidate.y < expanded.y + expanded.height && candidate.y + candidate.height > expanded.y`
       - Per Pitfall 4: the gap is NOT just "no pixel overlap" -- it's enforced as expanded bounding boxes
       - Returns true if ANY collision detected

    2. `findOpenDirection(center: Point, existingNodes: CanvasNodeInfo[]): number`
       - Per D-10: detect which direction has the most open space
       - Divide the space around center into 8 sectors (0, 45, 90, 135, 180, 225, 270, 315 degrees)
       - For each sector, count how many existing node centers fall within a 90-degree arc of that direction
       - Return the angle (in radians) of the sector with the fewest nodes
       - If no existing nodes (besides trigger), default to 0 (rightward)

    3. `computeOrbitalPlacements(triggerNode: CanvasNodeInfo, count: number, nodeSize: { width: number; height: number }, existingNodes: CanvasNodeInfo[], gap?: number): PlacementCoordinate[]`
       - Per D-09: generated nodes orbit the most recently edited node
       - Per D-10: multiple nodes fan out in the open direction
       - Call findOpenDirection to get the best angle
       - Spread `count` nodes across a 60-degree arc centered on the best angle
       - Starting radius = max(triggerNode.width, triggerNode.height) / 2 + gap + nodeSize.width / 2
       - Per D-12: for each placement, if collision detected, step outward by `gap` increments
       - Per Pitfall 3: maximum search radius = starting radius * maxSearchRadius (default 5). If exceeded, place at max radius in that direction (never loop infinitely)
       - Per D-11: use gap of 40px (from DEFAULT_SPATIAL_CONFIG.placementGap) for comfortable spacing
       - All coordinates are in canvas space (not viewport) per Pitfall 7
